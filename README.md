# 💰 AI Sales Prediction SaaS Dashboard

AI-powered Sales Prediction and Business Decision Support System built using Machine Learning and Streamlit.

This application predicts sales probability based on pricing, marketing, discounts, and customer data. It helps businesses understand potential sales performance and make better decisions using data-driven insights.

---

## 🚀 Features

- 🔐 User Login & Authentication
- 📊 AI Sales Prediction Model
- 📁 CSV Bulk Upload & Analysis
- 📥 Downloadable Reports
- 📈 Business Insight Dashboard
- ⚡ Real-time Prediction System

---

## 🧠 Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- FPDF (Reports)

---

```bash
pip install -r requirements.txt
